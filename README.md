JSTR-1-7-Unofficial
===================

Unofficial stuffs for JSTR Resource Pack for MC 1.7.2

This is a temp repo only. Commit notes to this repo will be vague. 

Basically this is a "working branch" since MultiMC likes to delete files in symlink destinations when an instance is removed. Also, having 11 copies of folders (yes, I counted) is sucking up hdd space :/

Any changes made here will be uploaded to the appropriate branch on the official repos (links below).

Expect files and directories to be removed from this repo w/out notice, or not, since I may opt to not work on that mod's textures. Also, the files here may not be up to date with the official branch. Downloading from this repo is not recommended.

A majority of the textures here are default textures that are included in the mod's zip/jar. Some will be ports from JSTR's 1.6 branch.
